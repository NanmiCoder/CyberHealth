
### 🚀 Cyber Health App Prompt

**角色设定：**
你是一位拥有 10 年经验的资深前端架构师和 UI/UX 设计师，擅长创造具有“赛博朋克（Cyberpunk）”和“未来科幻（Sci-Fi）”风格的高交互 Web 应用。

**任务目标：**
请使用 **React + Tailwind CSS + Lucide React** 开发一个单文件、可直接运行的**“每日康复伴侣 (Health Mate)” Web App**。

**核心内容与数据：**
这是一个针对“鼻炎-胃食管反流”共病患者的康复管理工具。请内置以下数据：
1.  **日程表**：早中晚及睡前的药物（艾司奥美拉唑、莫沙必利等）和行为（腹式呼吸、床头抬高）任务。
2.  **治疗方案**：详细的药物清单和行为守则表格。
3.  **病理闭环**：展示“鼻炎 → 口呼吸 → 吞气 → 胃胀 → 反酸 → 刺激咽喉 → 鼻炎加重”的恶性循环逻辑。

**UI/UX 设计要求（至关重要）：**

1.  **视觉风格 - 赛博医疗 (Cyber Health)：**
    * **背景**：深邃的午夜蓝/黑底色 (`#0B0F19`)，带有缓慢浮动的彩色光斑（光晕）和轻微的噪点纹理，营造沉浸感。
    * **卡片**：使用**深色玻璃拟态 (Dark Glassmorphism)**，半透明背景 + 细微的亮色边框 + 投影。
    * **配色**：使用高饱和度的霓虹色（青色 Cyan、翡翠绿 Emerald、紫罗兰 Violet、琥珀色 Amber）作为点缀和状态指示。字体使用无衬线体配合少量等宽字体 (Monospace) 增加科技感。
    * **入场动画**：页面元素加载时要有“故障风 (Glitch)”或“数据流加载”的入场效果。

2.  **交互与动效细节：**
    * **微交互 (Micro-interactions)**：点击任务卡片时，不要剧烈抖动，而是实现**极简的“微回弹 (Micro-bounce)”效果**（缩放 99% 再回弹），手感要像精密的机械按键。
    * **多巴胺奖励**：当用户勾选完成任务时，必须在屏幕中心触发**绚丽的烟花/粒子爆炸效果**（使用 Canvas 或简单的 DOM 粒子实现）。
    * **动态呼吸**：重要的按钮、状态标签和边框添加缓慢的“呼吸光效 (Pulse Glow)”。
    * **可视化闭环**：在“病理机制”板块，除了垂直流程图外，**必须绘制一条动态回流管线**（带流动的光点动画），从底部的“反酸”流回顶部的“鼻炎”，视觉上强调这是一个闭环系统。

3.  **响应式布局 (PC & Mobile)：**
    * **移动端**：单栏布局，底部悬浮导航栏（类似 iOS Dock），操作舒适。
    * **PC 端**：利用宽屏空间，采用**双栏或多栏网格布局**（Grid Layout）。例如任务列表左右分栏，治疗方案横向铺开。导航栏移动至顶部。

**技术约束：**
* **单文件输出**：所有代码（包括 CSS 动画、Canvas 逻辑、数据）必须包含在一个 `.jsx` 文件中。
* **无外部依赖**：不要使用 Three.js 或复杂的动画库，使用 React Hooks 和原生 CSS Keyframes 实现动效。
* **代码质量**：代码需模块化，逻辑清晰，确保无编译错误。
